盒子阴影

2016-04-24

简单的偏移阴影 --> box-shadow:5px 5px 0 0 rgba(0,0,0,0.8); 偏移模糊阴影 -- > box-shadow:5px 5px 5px 0 rgba(0,0,0,0.8); 外发光效果 --> box-shadow:0 0 5px 3px rgba(0,0,0,0.6); 内阴影 --> box-shadow:0 0 5px 3px rgba(0,0,0,0.6) inset; 盒子阴影可以嵌套，依次往后叠加，就是第一个是在最上层。-->box-shadow:0 0 0 1px rgba(204,0,0,1),0 0 0 4px rgba(0,146,10,0.4); 可以制作高光--> box-shadow:0 1px 0 0 rgba(204,0,0,1) inset; 高光、内阴影、外阴影-->background-color:#BE1C8B; box-shadow:0 6px 6px -2px rgba(255, 255, 255, 0.7) inset, 0 -5px 10px 3px rgba(0, 0, 0, 0.6) inset, 0 8px 10px 2px rgba(0, 0, 0, 0.3)
